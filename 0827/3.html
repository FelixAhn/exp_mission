태그선택자 : 태그 선택자는 HTML의 특정 태그 이름을 기반으로 스타일을 적용합니다. 예를 들어, HTML 문서의 같은 모든 요소에 같은 스타일을 적용하고 싶을 때 사용
클래스 선택자 : 클래스 선택자는 . (점) 기호 뒤에 클래스 이름을 사용하여 특정 클래스가 적용된 요소에 스타일을 적용, 여러 요소에 같은 스타일을 적용하고 싶을 때 유용

자손 선택자 : 부모 요소 안에 있는 특정 자손 요소에 스타일을 적용합니다. 예를 들어, .main 클래스 안의 p 태그에 스타일을 적용할 수 있습니다.

hover 선택자 : 사용자가 요소 위에 마우스를 올렸을 때 스타일이 변경
ID 선택자 : ID 선택자는 # (샵) 기호 뒤에 ID 이름을 사용하여 특정 ID를 가진 요소에 스타일을 적용 ID는 HTML 문서에서 고유해야 하며, 따라서 하나의 요소에만 적용
